    Github commands we used before
  • git commit -m название коммита("что бы другие разрабы поняли что мы изменили")- подготовка к загрузки на репозиторий
  • git push старт команды
  • git pull - скачать все файлы с репозитория и обновить
  • git add 'file name' будет один файл, если хотим все файлы тогда git add .
  • git status - cтатус того что происходит в данный моммент
